Web Development Journey

Building Dynamic Websites with Laravel & MySQL

My web development journey started when I learned how websites work beyond simple HTML and CSS pages. During my university projects, I began using Laravel and MySQL to develop more dynamic systems that could store, manage, and display data efficiently.

One of the projects that gave me valuable experience was developing a web-based system that involved user management, database integration, and CRUD functionalities. Throughout the development process, I learned how to structure databases, connect backend logic with user interfaces, and solve issues that appeared during testing.

Working with Laravel also improved my understanding of MVC architecture, routing, authentication, and system organization. Although there were many challenges along the way, each project helped me become more confident in building real-world web applications.

Mobile Development Journey

Creating Mobile Applications with Flutter & Firebase

After gaining experience in web development, I became interested in mobile application development and started learning Flutter. I was fascinated by how a single codebase could be used to create modern applications for multiple platforms.

Through my Final Year Project, SmartBite, I learned how to build complete mobile application features such as user authentication, profile management, meal recommendations, and database integration using Firebase.

During development, I faced many challenges, especially when handling Firestore data, designing user-friendly interfaces, and debugging application errors. However, these experiences taught me patience, problem-solving skills, and the importance of continuous learning in software development.

Today, Flutter and Firebase remain the technologies that I enjoy working with the most because they allow me to transform ideas into practical and interactive mobile applications.
